# Comparison: Disadvantages

<div class="grid grid-cols-2 gap-6 mt-4">

<div>
  <h3 class="text-lg font-bold mb-3 text-blue-600">Single Collection</h3>
  
  <v-clicks>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Requires filtering</div>
    <div class="text-sm text-gray-600">Must filter items by variable name</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Slower for large catalogs</div>
    <div class="text-sm text-gray-600">Loading all items can be slow</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Less intuitive</div>
    <div class="text-sm text-gray-600">Variables mixed together</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Generic metadata</div>
    <div class="text-sm text-gray-600">Can't have variable-specific collection info</div>
  </div>
  
  </v-clicks>
</div>

<div>
  <h3 class="text-lg font-bold mb-3 text-green-600">Per-Variable Collections</h3>
  
  <v-clicks>
  
  <div class="mb-3">
    <div class="font-semibold">❌ More complex structure</div>
    <div class="text-sm text-gray-600">Many collections to manage</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Harder multi-variable queries</div>
    <div class="text-sm text-gray-600">Must query multiple collections</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Metadata duplication</div>
    <div class="text-sm text-gray-600">Spatial extent repeated per collection</div>
  </div>
  
  <div class="mb-3">
    <div class="font-semibold">❌ Collection proliferation</div>
    <div class="text-sm text-gray-600">Can become unwieldy with many variables</div>
  </div>
  
  </v-clicks>
</div>

</div>
